# Memory 101

> *You don't replay the past — you reconstruct it. Every time you remember something, you are building it from scratch, and you can get it wrong.*

---

## The Big Picture

Memory feels like a video recording. You experienced something, it got stored, and when you remember it, you play the tape back.

This model is completely wrong.

Memory is **reconstructive**, not reproductive. Every time you retrieve a memory, you are actively rebuilding it from fragments — and in that rebuilding, it can be altered, contaminated, and distorted. This isn't a flaw; it's a feature. Flexible memory is more useful than rigid recording. But it has serious consequences for eyewitness testimony, therapy, and self-understanding.

---

## The Memory System: Not One Thing

Memory is not a single system — it's a collection of distinct systems that operate differently:

```
                        MEMORY
                           │
          ┌────────────────┼────────────────┐
          │                │                │
    Explicit          Implicit          Working
    (Declarative)   (Non-declarative)    Memory
          │                │
    ┌─────┴─────┐    ┌─────┴─────┐
    │ Episodic  │    │Procedural │
    │           │    │           │
    │ Semantic  │    │ Priming   │
    └───────────┘    │           │
                     │Conditioning│
                     └───────────┘
```

### Explicit (Declarative) Memory
Memories you can consciously recall and describe:
- **Episodic memory:** Personal experiences with time and place — your first day of school, what you had for breakfast
- **Semantic memory:** Facts and general knowledge — Paris is the capital of France, dogs are mammals

### Implicit (Non-declarative) Memory
Memories you use without conscious recollection:
- **Procedural memory:** How to ride a bike, type, drive. Stored in the basal ganglia and cerebellum. You can't easily verbalize it.
- **Priming:** Exposure to one stimulus changes response to a related one — seeing "bread" makes you faster to recognize "butter"
- **Classical conditioning:** Automatic learned associations (Pavlov's dogs)

### Working Memory
The mind's scratchpad — holding and manipulating information in the moment. Limited capacity (~4 items or "chunks"). Critically related to intelligence and academic performance.

## How Memories Form: The Three-Stage Model

### 1. Encoding
Information enters through perception. Not everything gets encoded — **attention is the gatekeeper**. You only form strong memories for things you attend to and process deeply.

**Levels of processing (Craik & Lockhart):**
- Shallow: just the sound of a word ("does this rhyme with cat?")
- Deep: semantic meaning ("does this word describe something living?")

Deeper processing creates stronger, longer-lasting memories. This is why reading for meaning beats re-reading for familiarity.

### 2. Storage and Consolidation
**Short-term memory** → **Long-term memory** through consolidation.

The **hippocampus** is the critical transfer station. New memories are initially hippocampus-dependent — damage to the hippocampus (like in patient **H.M.**, who had both hippocampi removed) prevents formation of new explicit memories while leaving old ones intact.

**Sleep is when consolidation happens.** During slow-wave sleep, the hippocampus "replays" daily experiences to the cortex, gradually transferring memories to distributed cortical storage. This is why pulling all-nighters is catastrophically bad for learning.

### 3. Retrieval
**Retrieval is reconstructive, not reproductive.** Key factors:

- **Context-dependent memory:** You remember better in the same context where you learned (scuba divers remember lists better underwater if they learned them underwater)
- **State-dependent memory:** Emotional or physiological state at encoding affects retrieval (memories encoded while happy are easier to retrieve when happy)
- **Retrieval cues:** Each memory is associated with many cues; more matching cues = better retrieval

## Why Memory Fails: The Seven Sins of Memory

Psychologist **Daniel Schacter** organized memory failures into seven "sins":

**Sins of omission (forgetting):**
1. **Transience** — memories fade over time (Ebbinghaus's forgetting curve)
2. **Absent-mindedness** — failures of attention during encoding (why you can't remember where you put your keys)
3. **Blocking** — temporary failure to retrieve (tip-of-the-tongue phenomenon)

**Sins of commission (distortion):**
4. **Misattribution** — remembering correctly but assigning wrong source ("I thought of that idea!" — but you heard it from someone)
5. **Suggestibility** — memory altered by leading questions, misinformation
6. **Bias** — current knowledge and beliefs distort memories of the past
7. **Persistence** — unwanted memories that won't go away (trauma, intrusive thoughts)

## The Misinformation Effect

**Elizabeth Loftus** conducted landmark experiments showing memory can be contaminated by post-event information.

In her classic study: people watched a car accident, then were asked either "how fast were the cars going when they *hit* each other?" or "when they *smashed* into each other?" The "smashed" group reported faster speeds — and a week later, more of them falsely remembered broken glass (there was none).

**Implications:**
- Eyewitness testimony is far less reliable than juries believe
- Leading questions from therapists or interviewers can implant false memories
- **False memory syndrome** — people can be convinced they have memories of events that never happened

> **Aha! moment:** A memory feeling vivid and certain tells you almost nothing about its accuracy. Confidence and accuracy are largely uncorrelated in eyewitness memory.

## Emotional Memory

The **amygdala** enhances encoding of emotional events — especially threatening ones. This is why you remember where you were on 9/11 but not what you had for breakfast last Tuesday.

**Flashbulb memories** — vivid, detailed memories of emotionally significant events — feel more accurate than ordinary memories. Research shows they're not actually more accurate, just more confidently held.

The upside of emotional memory enhancement: it helps you learn from important experiences.
The downside: traumatic memories are over-consolidated, intrusive, and resistant to normal forgetting — the mechanism behind PTSD.

## How to Improve Memory

**Spaced repetition:** Practice at increasing intervals (study, wait, test yourself, wait longer, test again). The "spacing effect" is one of psychology's most replicated findings — exponentially more effective than massed practice.

**Retrieval practice (testing effect):** Testing yourself on material is more effective than re-reading or re-studying. The act of retrieving a memory strengthens it more than the act of encoding it again.

**Interleaving:** Mixing different types of problems during study (math, science, language) produces better retention than blocked practice, even though it feels harder.

**Sleep:** Non-negotiable for memory consolidation. Naps after learning significantly improve retention.

**Elaborative encoding:** Connect new material to what you already know. Build a rich web of associations. Mnemonics work because they force this.

---

## Key Terms
- **Hippocampus:** Brain structure critical for forming new explicit memories
- **Consolidation:** Transfer of short-term memories to long-term storage, primarily during sleep
- **Encoding:** The initial processing of information into memory
- **Retrieval:** Reconstruction (not playback) of stored information
- **Misinformation effect:** Memory contamination by post-event information
- **Spaced repetition:** Reviewing material at increasing intervals — the most effective study technique
- **Retrieval practice (testing effect):** Testing yourself strengthens memory more than re-studying
